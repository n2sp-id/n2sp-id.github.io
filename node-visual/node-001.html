Node Visual 001

Bandung · Event Komunitas · Dokumentasi Visual

Node ini merekam aktivitas ngopi santai yang berlangsung di ruang publik Bandung, di mana fotografer, model, dan kreator visual berkumpul tanpa agenda kaku. Kamera hadir bukan sebagai alat produksi, tetapi sebagai medium observasi.

Percakapan visual tumbuh secara organik — dari meja kopi, gestur tubuh, hingga interaksi spontan dengan ruang sekitar. Praktik ini menjadi ciri khas pendekatan N2SP dalam membangun arsip visual komunitas.

Node Visual ini tidak berfungsi sebagai galeri karya, melainkan sebagai catatan konteks: siapa yang hadir, di mana aktivitas berlangsung, dan bagaimana visual terbentuk.

Ke depan, dokumentasi visual dari node ini akan terhubung dengan Instagram, blog, dan Google Maps sebagai bagian dari ekosistem digital N2SP berbasis G-Loop Method.
